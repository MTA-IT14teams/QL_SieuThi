Hướng dẫn sử dụng

1.Đầu tiên,các bạn phải kết nối Server, điền đầy đủ thông tin :

  • +Server Name
  • +Authemtication (Có 2 chế độ SQL Server Authentication OR Windows Authentication)
  • +Login ,Password(Nếu dùng chế độ SQL Server Authentication )
  • +Database Name

2.Sau khi kết nối thành công,bạn phải đăng nhập tài khoản đã được cấp phát.

3.Sau khi đăng nhập thành công màn hình chính xuất hiện, có 5 modul cơ bản của hệ thống :Trang chủ, Quản Trị Hệ Tống, Quản Lí,Giữ Liệu Thống Kê, Trợ Giúp, Giới Thiệu

4.Khi ấn vào Modul Quản Lí, chọn Nhân Viên,có 5 chức năng chính:

  • +Thêm Nhân Viên
  • +Sửa thông tin Nhân Viên
  • +Xóa Nhân Viên
  • +Refresh(Cập nhật) Nhân Viên
  • +Tìm Kiếm Nhân Viên


5.Khi ấn vào Modul Quản Lí, chọn Khách Hàng,có 7 chức năng chính:

  • +Thêm Khách Hàng
  • +Sửa thông tin Khách Hàng
  • +Xóa Khách Hàng
  • +Refresh(Cập nhật) Khách Hàng
  • +Lưu Khách Hàng
  • +Hủy Khách Hàng
  • +Tìm Kiếm Khách Hàng


6.Khi ấn vào Modul Quản Lí, chọn Hàng Hóa,có 7 chức năng chính:

  • +Thêm Hàng Hóa
  • +Sửa thông tin Hàng Hóa
  • +Xóa Hàng Hóa
  • +Refresh(Cập nhật) Hàng Hóa
  • +Lưu Hàng Hóa
  • +Hủy Hàng Hóa
  • +Tìm Kiếm Hàng Hóa


7.Khi ấn vào Modul Quản Lí, chọn Bán Hàng,có 2 chức năng chính:

  • +Bán
  • +Hủy


8.Khi ấn vào Modul Quản Lí, chọn Kho Hàng,có 5 chức năng chính:

  • +Thêm Kho Hàng
  • +Sửa thông tin Kho Hàng
  • +Xóa Kho Hàng
  • +Refresh(Cập nhật) Kho Hàng
  • +Tìm Kiếm Kho Hàng
